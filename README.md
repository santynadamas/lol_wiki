## 📖 About

**LoL Wiki** is a personal project built to bring together everything a League of Legends player needs in one place — champion stats, abilities, item builds, patch notes, and a personalized experience through user accounts and favorites. No more tab-juggling between five different sites. Just one clean, fast wiki.

---

## ✨ Features

<table>
<tr>
<td width="50%" valign="top">

### 🧙 Champion Database
Browse the full roster with detailed pages for every champion, including **all abilities**, broken down clearly.

### 🔍 Smart Search & Filters
A fully functional search bar, plus filtering by:
- Name
- Tier
- Win rate
- Class — Tank 🛡️ · Mage 🔮 · Fighter ⚔️ · Assassin 🗡️ · Support 💚 · ADC 🏹

</td>

### 🛠️ Item Builds
Explore recommended item builds tailored to each champion.

### 📰 Patch Notes & News
Stay current with the latest balance changes and game updates.

### 👤 User Accounts
Sign in with **Google** or a **username & password**.

### ⭐ Favorites
Save your favorite champions, skins, and items right to your profile.
