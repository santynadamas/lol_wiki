# LoL Wiki

A complete and up-to-date League of Legends wiki featuring champions, abilities, items, builds, patch notes, and game guides.

**Live demo:** [lol-wiki-dusky.vercel.app](https://lol-wiki-dusky.vercel.app)

## About

LoL Wiki is a personal project built to bring together everything a League of Legends player needs in one place: champion stats, abilities, item builds, patch notes, and a personalized experience through user accounts and favorites.

## Features

- **Champion database** — Browse the full list of champions with detailed pages for each one, including all abilities.
- **Search and filters** — A fully functional search bar, plus filters by name, tier, win rate, and champion class (Tank, Mage, Fighter, Assassin, Support, ADC).
- **Item builds** — Explore recommended item builds for champions.
- **Patch notes & news** — Stay up to date with the latest balance changes and game updates.
- **User accounts** — Sign in with Google or a username and password.
- **Favorites** — Save your favorite champions, skins, and items to your profile.

## Tech Stack

- **Framework:** Next.js (TypeScript)
- **Authentication:** NextAuth.js (Google OAuth + Credentials)
- **Database:** MongoDB
- **Deployment:** Vercel

## Status

This project is under active development. Some features may continue to evolve as new patches and content are added.
